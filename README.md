# movies-explorer-frontend

## О проекте
Данный проект - это frontend для сервиса поиска фильмов.

## Описание
Данный проект - дипломная рабта по курсу Web-разрабтка от Яндекс.Практикума.

## Стек технологий:
- React
- React hooks
- Figma
- JSX

## Запуск проекта
npm start — запускает проект на http://localhost:3000
